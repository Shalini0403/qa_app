Simple RAG Application with PDF Upload
A Retrieval-Augmented Generation (RAG) application that allows users to upload PDF documents, ask questions, and get AI-generated answers based on the document content. The application uses FAISS for efficient document retrieval and OpenAI's GPT models for text generation.
Features

PDF document upload and text extraction
Document indexing using FAISS for efficient retrieval
Question answering using OpenAI's GPT models
Streamlit web interface
Secure configuration management

Prerequisites

Python 3.8 or higher
OpenAI API key
Sufficient OpenAI API credits
